# 컴퓨터 통신

> 컴퓨터는 어떻게 데이터를 주고 받을까?

## 1. 정보통신

### 통신 Communication

한 지점에서 원하는 다른 지점까지 의미 있는 정보를 보다 빠르고 정확하게 상대방이 이해할 수 있도록 전송하는 것

- 봉화
- 전신 telegraphs
  - 프랑스 혁명기 클로드 샤프가 발명
  - 가시거리 있는 다른 탑에 신호 전송
  - 전쟁, 날씨 등 신속한 정보 전송 수단
  - 인코딩, 디코딩은 처음과 끝에 이루어짐; 중간에는 전달만
  - 비, 안개, 전신수 부주의 등으로 인한 오류가 빈번
- **_전기 전신_**; 1830년대
  - 전기의 발견; 전류, 전압, 저항(옴)
  - 전기 전신을 상용화(1844)
  - 미국 새무얼 모스, 알프레도 베일, 영국 윌리엄 쿡, 찰스 휘트스톤이 거의 동시에 발명
  - 날씨, 신문, 표준시간 등 정보전달 혁명
  - 시간과 공간 소멸
  - 코드 사용
  - 메시지 전달; 비밀 유지와 간결성 필요, 특히 국제 전신은 경제성이 더 중요
  - 발신자, 수신자 간 사전협의 필요; 암호학, 코드북 사용
  - 현대 정보이론의 시초가 됨

### 전기 통신 역사

전기 통신 == 원거리 통신

- 1844 모스; 철도 따라 전선 연결, 최초의 전보
- 1876 벨; 선으로 연결된 전화 발명
- 1901 마르코니; 모스 부호, 무선 송출기 발명하여 원거리 통신, 특히 선박, 산악 등에 활용하여 무선 통신의 막을 올림

### 컴퓨터 통신

정보통신 시스템

- 정보 전송 기술과 컴퓨터에 의한 정보 처리 기술을 결합해 정확한 정보를 원하는 사람들에게 전하는 것
- 통신 회선에 연결된 하나 또는 그 이상의 컴퓨터나 단말기를 이용한 정보의 전달을 말함

통신의 구성요소

- 정보원: 송신자, 수신자, 다양한 단말 장치
- 전송매체: 전화선, 전파, 공기 등
- 프로토콜: 통신에 필요한 규칙

### 컴퓨터 네트워크

- 컴퓨터 상호 간 정보 교환과 정보 처리를 위한 데이터 통신망
- 2대 이상의 컴퓨터를 도로망과 같이 케이블 등으로 연결
- 정보 전송은 네트워크 프로토콜 또는 통신 규약을 따름
- 인터넷 등

국제 표준: ISO, CCITT 등에서 제정

### 컴퓨터 통신의 발전

- 1960's 후반: 대형 컴퓨터-터미널들 간 온라인 이용, 미 국방부 ARPANet
- 1970: 컴퓨터 간 통신 필요성 증대로 통신 장비 및 통신 SW 개발, Ethernet, 고도의 조직적 네트워크 필요성 대두(원거리 통신망 출현 배경)
- 1980: 기기들 간 신뢰성 있는 정보를 교환할 수 있는 체계의 필요성 증대로 TCP/IP 도입, 근거리 통신망이 등장하게 되는 배경 형성
- 1990: 인터넷 급격한 성장, WWW를 중심으로 한 인터넷 서비스 일반화, ISDN, ADSL 등 새로운 방식의 디지털 연결 방식 연구
- 2000: ISDN, ADSL 등 인터넷 연결 방식 상용화, 무선 네트워크 사용 확산, IMT-2000 등 새로운 통신방식 연구, 각 가정에 초고속 통신망 보급, 멀티미디어 기반 네트워크 구축
- 2010: 스마트폰/태블릿 PC 통한 지능적 무선 통신 활성화, 초고속 LTE 및 LTE-A 통신 실용화
- 2020: 초연결hyper-connected의 5세대 이동통신 개시 예정, IoT를 위한 다중연결성, 효율성 향상

### 컴퓨터 통신 용어

- 프로토콜 Protocol: 통신과 관련된 형식과 절차에 따른 규정, 메시지 종류와 형식, 메시지 교환 절차, 흐름제어, 에러제어 등
- 대역폭 Bandwidth: 한 선(채널) 통해 전달되는 초당 정보의 양, 단위는 bps(bits per second)
- 모뎀 MODEM: 컴퓨터 통신 데이터는 디지털, 전화선 데이터는 아날로그, 필요에 따라 디지털 또는 아날로그 신호로 변환시켜주는 장치
- DTE(Data Terminal Equipment): 통신망에서 네트워크 끝에 붙은 장비들 총칭, 일반적으로 컴퓨터 지칭하나 프린터, 중계기, 단말기 등 통틀어 부름

### 네트워크 구성요소

서로 다른 정보 기기들 사이에 정보를 주고받기 위하여 송수신 정보기기, 전송 매체인 케이블 등으로 구성

- 하드웨어

  - 단말 장치: 서버PC, 클라이언트PC, 스캐너, 팩시밀리, 스마트폰
  - 전송 매체: 케이블
  - 접속 장치: LAN 카드
  - 접속 집중 장치(중계/통신 장치): 허브, 라우터, 스위치, 게이트웨이 등

- 네트워크 전송 매체

  - 송신 측-수신 측 연결해 주는 물리적인 경로, 컴퓨터 네트워크를 구성하는 노드들 사이를 회선으로 연결하는 데 사용
  - 유선 전송 매체
    - 선으로 직접 연결하는 매체
    - 트위스트 페어 케이블(연선): 두 개의 선을 나선형으로 꼬아서 플라스틱으로 코팅하여 외부 간섭을 최소화, 전송 속도나 거리에 제한을 받지만 가격이 저렴하여 전화선, 근거리 통신망 등에 많이 사용
    - 동축 케이블: 중앙에 있는 심선을 절연체로 감싸고, 그 둘레를 다시 그물 모양 금속 끈 막으로 감싼 다음 플라스틱 물질로 피복 입힌 케이블, 구리선을 싸고 있는 피복으로 인해 외부 간섭을 덜 받게되며 전력 손실이 적음, 장거리 전화망, CATV
    - 광섬유 케이블: 전기적인 신호 대신 빛의 형태로 전송하는 얇은 유리 섬유로 구성된 케이블, 초고속 전송, 대용량 전송에 적합, 충격/잡음/누화 등 전자기 간섭에 영향을 받지 않기 때문에 매우 안전한 전송매체, 전화망/데이터망/종합 통신망 등으로 가장 널리 활용
  - 무선 전송 매체
    - 공간에서 전자기파를 매체로 이용하여 정보를 송수신하는 방식, 안테나가 정보 송수신 담당
    - 라디오 파: 10~100m 내 다방향성 무선파로 간섭 효과가 나타나는 단점, 블루투스는 10m 내 단거리에서 이동전화/노트북 등 휴대 기기를 서로 연결해 정보를 교환하는 개인용 네트워크(PAN)에 초점을 두어 개발된 근거리 무선 통신의 표준
    - 마이크로 파: 유선 전송 매체를 설치하기 어려운 강/늪/사막 등이나 장거리 통신에 활용, 전자파 에너지를 잘 집중시키기 위해 파라볼라 안테나 이용, 전송은 고속이고 비용이 적게 들어 장거리 통신/이동전화/국제전화 등에 활용되나 장애물이나 기상 상태에 따라 감쇄 현상 발생
    - 위성 마이크로파: 지구 주위를 선회하는 통신위성 통해 지상 중계소와 통신할 때 사용, 거리/지형에 상관없이 넓은 범위에 데이터 전송, 위치 확인 시스템은 위성 통신 이용해 자신의 위치 알 수 있는 시스템
    - 적외선: IrDA 무선 통신, 디지털 카메라에서 처음 사용한 무선 통신, 디지털 카메라에서 프린터로 데이터를 바로 전송해 인쇄할 때 활용, 리모컨/무선 키보드/무선 마우스에 이용, 가까운 위치에 있어야 하고 전송 속도 느림

- 네트워크 장비

  - **네트워크 인터페이스 카드 Network Interface Card, NIC**: 모든 형태의 네트워크와 컴퓨터를 연결해 주는 장치, 랜카드/이더넷 카드 등으로 불림, 속도와 접속 방식에 따라 여러 종류로 나눔
  - **허브 Hub**: 데이터가 하나 또는 그 이상 방향으로부터 한 곳으로 모이는 장소, 들어온 데이터들은 다시 하나 또는 그 이상의 방향으로 전달됨, **스위칭 허브**는 들어온 데이터를 어떠한 목적지까지 결정하여 전달할 수 있음(효율적)
  - **라우터 Router**: 전 세계 수많은 네트워크 중 목적지까지 데이터를 보내고 받을 수 있도록 하는 기능을 하는 하드웨어, 주위 라우터들과 통신을 하거나 또는 미리 주어진 정보를 가지고 어떠한 목적지까지 가는 최적의 경로 제공

## 2. 컴퓨터 네트워크

> 여러 대의 컴퓨터가 서로 통신하기 위해서는 어떻게 해야 할까?

### 통신 방식

통신 방식

- 각 DTE간 통신 방식은 자료 흐름 방향과 동시성 여부에 따라 구분
- 단방향 통신 simplex: 한 DTE는 송신 기능만 가능, 다른 DTE는 수신 기능만 가지는 형태, TV, 라디오 등 방송용
- 반이중 통신 half duplex: 통신하는 두 DTE가 시간적으로 교대로 데이터 교환 방식, 무전기 등
- 전이중 통신 full duplex: 두 DTE가 동시에 송수신 가능한 형태, 송신과 수신 위한 독립적인 채널 필요, 전화 등

통신망

- 통신 시스템들을 통신 회선으로 연결해 통신이 가능하도록 한 시스템

- 소유자에 따라 구분

  - 사설 통신망
  - 공중 통신망

- 네트워크 접속에 따른 분류: 비트를 어떻게 전송할 것인가

  - 회선 교환 방식: 전화망과 같은 원리, 폐쇄회로 네트워크(한번 연결이 이뤄지면 다른 가입자들에 대해 연결이 폐쇄되기 떄문)
  - 축적 후 교환 방식
    - 패킷 교환 방식 : 한 회선으로 여러 대 컴퓨터에서 동시에 통신 가능, 데이터는 패킷으로 불리는 단위로 전송, 패킷은 헤더(데이터가 전달될 곳의 주소와 패킷 순서 데이터)-테일러(오류 수정하기 위한 정보)로 구성
      - 데이터그램 방식
      - 가상 회선 방식
      - TDMA, Time Division Multiple Access; 패킷을 정해진 일정 시간 동안만 전송, 시분할 다중 접속 방식, 사용자가 거의 동시에 네트워크 사용하는 효과, 접속이 많아지면 이것도 동시 접속 불가능해짐
      - ALOHA; 1970년 하와이 대학에서 구현된 최초의 임의 접근 네트워크, 2개 이상 DTE가 데이터를 보내 충돌 발생할 경우 중앙에서 감지하여 충돌 생긴 두 DTE에 통보하고, 두 DTE는 임의의 수를 발생시켜 임의의 시간 동안 기다렸다가 다시 데이터 전송
      - Ethernet CSMA/CD: 데이터를 전송하기 전 회선이 사용 중인지 감시하고 있다가 회선이 비어있을 때 데이터 전송, 만약 데이터 전송하는 시점에 다른 장치가 동시에 전송을 개시하면 충돌 발생, 충돌한 데이터들은 버려지고 데이터를 전송한 장치들에게 재전송을 요구, 각 장치들은 일정 시간을 대기한 후 성공할 때까지 어느 횟수만큼 데이터를 재전송, 가장 광범위하게 설치된 LAN 기술로 IEEE 802.3 국제 표준화 기구에 의해 표준으로 정의됨
    - 메시지 교환 방식

- 거리에 따라 구분

  - 근거리 통신망(LAN, Local Area Network): 초기 300m 이하 통신회선으로 연결된 컴퓨터간 네트워크 의미하고 **이더넷** 기술 사용, 현재는 거리 개념보다는 어떤 목적을 위한 네트워크 집합 의미하며 좀 더 고속화된 ATM이나 무선기술 등 쓰임

    - 특징

      - 고품질 전용 회선을 별도로 설치하여 사용
      - 전송 속도가 매우 빠름; 10Mbps ~ 100Mbps 이상
      - 고품질 선로로 에러율 낮음; 10 ~ 12 이하
      - 거리가 가깝기 때문에 전송 지연 시간이 낮음
      - 컴퓨터 이외에도 프린터 서버 같은 장비들과의 자유로운 연결 제공

    - 응용

      - 물리적 자원 공유: 프린터 등 하드웨어 연결, 여러 컴퓨터가 공유 가능
      - 파일 전송 기능: LAN 이용한 파일 전송, 같은 자원 중복 없이 집중식 관리 가능, 공동 작업시 데이터 일관성, 관리에 도움
      - 처리기 공유: 다른 컴퓨터에 로그인하여 작업 간으
      - e-mail 기능: 메일 기능 사용해 사용자간 통신 및 문서 전송, 공지사항 열람 등 통해 시간 절약 가능
      - 외부망 접속 기능: 하나의 컴퓨터만 외부와 연결되어 있어도 그 컴퓨터를 통해 LAN 상 모든 컴퓨터가 외부와 통신 가능

    - 구성형태

      - Bus 형: 버스라는 하나의 공통 선에 각 노드들이 접속된 형태, 근본적으로 방송 특성 가지므로 여러 노드가 동시에 전송 원할 때 충돌 문제 해결할 수 있는 적절한 방법 필요, 일반적인 LAN 구성 형태
      - Ring 형: 각 장치들이 원형을 이루는 신호 경로를 따라 두 개 이상의 다른 장치와 연결, 전체적으로 링 모양 가지는 네트워크, 정보 흐름은 단방향으로만 진행, 장애 발생할 경우 제어 장치가 링으로부터의 흐름 가로채면서 관리
      - Star 형: 중앙 제어 노드 중심으로 각 노드 연결은 모두 중앙 제어 장치를 통해 이루어짐, 한 곳에 데이터 집중되므로 이를 관리하는 시스템이 전체 성능에 큰 영향, 관리가 쉬움

  - 원거리 통신망 (WAN, Wide Area Network)

    - 넓은 지역에 걸쳐 분포하는 많은 교환 노드들과 DTE들로 구성됨
    - LAN이 보편화되면서 WAN은 주로 LAN-LAN 연결하는 수단으로 사용
    - 통신 서비스 회사에 의해 공중망 형태로 운영

## 3. 유/무선 통신

> 5G는 얼마나 빠를까?

### 유무선 통신 방식

ISDN (Integrated Service Digital Network)

- 기존 전화망을 통해 음성 정보(전화)와 데이터를 동시에 보내고자 하는 개념
- Integrated Service는 통합 서비스로서 기존 음성과 디지털 신호를 하나의 전화망 서비스를 통해 제공
- 기존 모뎀과는 달리, 전화와 데이터 전송을 위한 독립된 채널로 전화 사용과는 상관없는 통신 가능

ADSL (Asymmetric Digital Subscriber Line)

- 비대칭 디지털 가입자 회선의 약자
  - 일반적으로 송신 데이터보다 수신 데이터가 많으므로, 대역폭을 수신에 더 많이 할당
- 음성 통화와 고속 인터넷 통신이 동시에 가능한 차세대 접속 서비스
- 기존 전화선으로 대역 나누어 음성과 데이터 동시 전송
- 아직 외국에선 많이 쓰임

Bluetooth

- 1994, 에릭슨이 최초 개발한 PANs(Personal Area Networks)를 통한 산업 표준
- IEEE 802.15.1 규격 사용
- 전 세계적으로 사용할 수 있는 무선 주파수 이용하여, 서로 다른 다양한 장치들이 안전하고 저렴한 비용으로 통신 가능하게 함

WiFi

- IEEE 802.11 기반 무선 랜 연결, 장치간 연결, PAN/LAN/WAN 구성 등 지원
- 무선 접속 장치가 설치된 곳으로부터 몇 미터~몇 백 미터 거리 내 고속 인터넷을 무선으로 연결할 수 있는 근거리 통신망 기술

### 이동 통신 (Mobile Communication)

- 사용자가 자유롭게 이동하는 중에도 계속적인 통신이 가능하도록 해주는 통신 시스템
- 무선 통신 기술 사용

1세대 이동 통신 (First Generation, 1G)

- 아날로그 타입의 cellular 방식 사용
  - 서비스 지역을 작은 크기의 셀들로 분할, 셀 내 사용자들은 셀 중앙에 설치된 기지국과 통신하는 구조의 이동통신 시스템
- 음성 통화만 가능

2세대 이동 통신 (Second Generation, 2G)

- 퀄컴사가 개발, 디지털 방식 코드 분할 다중화 접속 방식(CDMA)
- CDMA 2000은 CDMA 보다 폭넓은 주파수를 사용하기 때문에 정보를 보다 빠르고 효율적으로 송수신할 수 있으므로, 인터넷 데이터, 비디오, CD 수준 음악을 빠르게 전송할 수 있게 해줌

3세대 이동 통신 (Third Generation, 3G)

- 1990년대 말부터 문자, 음성, 동화상 등 멀티미디어 데이터를 현재 유선 인터넷과 비슷한 화질로 전송할 수 있는 특징
- IMT-2000(International Mobile Telecommunication-2000) 포함, 2GHz 주파수 대역 이용해 2~2.4Mbps 속도 제공

4세대 이동 통신 (Fourth Generation, 4G)

- 하나의 단말기로 위성망, 무선랜, 인터넷 모두 사용
- 2011년 7월 도입된 LTE
- 2012년 1월 국제전기통신연합은 LTE-advanced와 Wibro-advanced를 4G 국제표준으로 채택

5세대 이동 통신 (Fifth Generation, 5G)

- 초당 1기가 비트 이상의 데이터가 전송되는 초고속 통신 서비스
- 사양
  - 최대 1 ~ 10 Gbps의 초고속 데이터 처리 속도
  - 1ms 이하의 저지연과 품질 안정성
  - 수많은 디바이스와 다중연결성
  - 확장, 운용, 진화가 용이한 유연성, 저비용의 경제성
- 핵심 가치
  - 하드웨어적 발전
  - 고객 경험
  - 연결성
  - 지능화
  - 효율성
  - 신뢰성

이동 통신 세대별 속도 비교

- 1G: 14.4kbps
- 2G: 144kbps
- 3G: 14Mbps
- 4G: 75Mbps
- 5G: 1Gbps
