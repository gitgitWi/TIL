### `index.js`

간단한 백엔드 애플리케이션 만들기

```json
  "scripts": {

  },

```

package 실행

```bash

$ npm run start
```

## 04. MongoDB Model & Schema

회원가입 시 필요한 내용


### model

schema 를 감싸주는 역할

User model

- models/User.js

```js

const mongoose = require("mongoose");

// 일반 SQL에서 table 생성하는 것과 조금 차이 있음
const userSchema = mongoose.Schema({
    name: {
        type : String,
        maxlength: 50,
    },
    email: {
        type: String,
        trim: true,
        unique: 1
    },
    password: {
        type: String,
        minlength: 5
    },
    lastname: {
        type: String,
        maxlength: 50
    },
    role: {
        type: Number,
        default: 0
    },
    image: String,
    token: {
        type: String
    },
    // token 의 만료기간 지정
    tokenExp: {
        type: Number
    }
});

// 위에서 생성한 Schema를 model 로 감쌈
// model("NAME", SchemaName)
const User = mongoose.model("User", userSchema);

// User 모델을 다른 곳에서도 사용할 수 있도록 export
module.exports = { User }

```
